Groupy Alarm

Groupy Alarm is like a regular Apple alarm app but on steroids. Create multiple alarms; organize them in groups. Groupy Alarm is also a relaxing app with 20 relaxing sounds. Set a timer a, relax.

What I Used:
  • Swift
  • UIKit
  • CoreData
  • AVAudioPlayer
  • MVVM/MVC architecture
  • 100% Programatic UI

CINEXT

CINEXT is a movie app to find Movies and TV Shows and create To Watch and Watched Lists. CINEXT gets data from themoviedb api.

What I Used:
  • Swift
  • UIKit
  • Firebase
  • YoutubePlayer-in-WKWebView
  • MVVM/MVC architecture
  • 100% Programatic UI

My Locks

My Locks app is a project that combines software and hardware. It connects to an arduino board/system that simulates a door lock. Control the lock via the iOS and watchOS app.

What I Used:
  • Swift
  • UIKit
  • Firebase
  • CoreBluetooth
  • WatchKit
  • Push Notifications
  • MVC architecture and Storyboards
